# 🕹️ Sewages&Punks: Консольная игра подземелье

>[!NOTE]
>### Проект представляет собой процедурно-генерируемый Dungeon Crawler в CLI интерфейсе, вдохновленный эстетикой киберпанка.

### 🛠 Архитектура проекта

<details>
<summary>Нажмите, чтобы развернуть описание</summary>

    Проект разделен на логические модули для удобства расширения:

    src/core.py — ядро игры, механики перемещения, взаимодействия и боя.
    src/businesslogic_*.py — скрипты для генерации карты и игровой логики.
    src/display.py — визуализация интерфейса.
    src/entities.py — описание игрока и противников.
    src/localization.py — текстовые ресурсы.
    src/constants.py — константы.
    pregen_levels/ — заранее созданные не сгенерированные уровни.

</details>
         
### ✅ Реализовано: 

<details>
<summary>Нажмите, чтобы развернуть описание</summary>


- [x] Procedural Generation: Генерация случайных уровней.
- [x] Survival System: Здоровье, ловушки (с механикой разминирования), сундуки.
- [x] Combat System: Пошаговые сражения с врагами.
- [x] Game States: Главное меню, система сохранений/загрузки прогресса.
- [x] Exits: Ключи и двери для выхода из секторов и генерации новых.
- [x] Immersive: Вступление и сцена окончания игры.
    
</details>

### 🎮 Как запустить
Для работы игры в исходном виде (через код) вам понадобится установленный Python 3.10 или выше.

<details>
<summary>Нажмите, чтобы развернуть описание</summary

Шаг 1. Клонирование репозитория

    Откройте терминал (или командную строку) и скачайте проект к себе на компьютер:
    
    git clone https://github.com/Icy-beep/Sewages-Punks
    
    Если у вас не установлен Git, вы можете просто скачать архив проекта кнопкой 
    Code -> Download ZIP и распаковать его.

Шаг 2. Переход в директорию проекта
        
    В терминале перейдите в папку, где лежит файл main.py:

    cd Sewages-Punks
        

Шаг 3. Создание виртуального окружения
        
    Рекомендуется использовать виртуальное окружение, 
    чтобы зависимости проекта не конфликтовали с системными:

    
    Для Windows:
        
        1. python -m venv venv
        2. venv\Scripts\activate

    Для Linux / macOS(Внимание! Возможна некорректная работа. Игра разрабатывалась под Windows):
        
        1. python3 -m venv venv
        2. source venv/bin/activate


Шаг 4. Запуск игры

    Запустите главный файл скрипта:

        Для Windows:
        python main.py

        Для Linux / macOS(Внимание! Возможна некорректная работа. Игра разрабатывалась под Windows):
        python3 main.py

</details>

### 🗺 Дорожная карта (Roadmap)

<details>
<summary>Нажмите, чтобы развернуть описание</summary

Проект развивается в свободном темпе. Вот что планируется добавить в будущем:

- [ ] **Лор:** Расширить вступление, добавить сюжет, прописать персонажей.
- [ ] **ИИ для противников:** Сделать ИИ чтобы враги перемещались по карте.
- [ ] **Туман войны:** Реализация динамического радиуса обзора, расширяемого предметами, способностями, имплантами.
- [ ] **Система имплантов:** Кастомизация способностей через био-модификации.
- [ ] **Терминалы и Взлом:** Интерактивные объекты на карте для получения данных о лоре или удаленного открытия дверей.
- [ ] **Визуальные эффекты:** Внедрение глитч-эффектов, расширенной цветовой палитры и имитации динамического освещения в консоли.
- [ ] **Улучшения противников:** Добавить им особые умения сделать каждого противника уникальным.
- [ ] **Звуковой движок:** Атмосферный саундтрек для полного погружения.

</details>
