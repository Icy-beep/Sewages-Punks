# Проект: Консольная игра “Подземелье”
````
Игра выполнена в сеттинге
            _                                 _    
           | |                               | |   
  ___ _   _| |__   ___ _ __ _ __  _   _ _ __ | | __
 / __| | | | '_ \ / _ \ '__| '_ \| | | | '_ \| |/ /
| (__| |_| | |_) |  __/ |  | |_) | |_| | | | |   < 
 \___|\__, |_.__/ \___|_|  | .__/ \__,_|_| |_|_|\_\
       __/ |               | |                     
      |___/                |_|            
````
         
##### В данный момент реализованы: 
````diff
+ Генерация подземелья, карта подземелья, враги, здоровье врагов/игрока.
+ бой, ловушки, сундуки, разминирование ловушек, ключ к выходу, выход.
````

##### В данный момент не реализованы:
````diff
- Сохранение/загрузка игры.
- не до конца доделана "графика" UI - локализация/text, получение опыта/очков.
- особые навыки игрока и врагов, лорно - сюжетная составляющая.
````
